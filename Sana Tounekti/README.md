**Étudiante :** Sana Tounekti

## Description

SmartCare est une application mobile de gestion de santé intelligente développée avec Flutter. Elle permet le suivi des médicaments, la gestion des rappels, le suivi des activités quotidiennes, et l'assistance aux patients via un tableau de bord interactif.

## Technologies utilisées

- **Framework :** Flutter (Dart)
- **Backend :** Firebase (Auth, Firestore, Storage, Messaging)
- **State management :** Provider
- **AI :** OpenRouter AI
- **Plateformes :** Android, iOS, Web

## Fonctionnalités principales

- Gestion et rappel de médicaments
- Tableau de bord patient et assistant
- Suivi des activités quotidiennes
- Jeux de stimulation cognitive
- Calcul IMC
- Alertes d'urgence
- Gestion de documents patients
- Liens familiaux
- Multilingue (français, anglais, arabe)
